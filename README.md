# dzokha.vn
1. cài đặt Node.js
https://nodejs.org/en/download
```
node -v
npm -v
```

note: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

1. Cài đặt lại thư viện (Nếu cần)
```
npm install
```
# Hoặc nếu bạn dùng yarn
```
yarn install
```

2. Biên dịch dự án (Build)
```
npm run build
```
# Hoặc
```
yarn build
```

3. Kiểm tra bản build ngay tại máy (Preview)
```
npm run preview
```
# Hoặc
```
yarn preview
```
